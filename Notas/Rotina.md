## Verificações Matinais do Sistema (Pós-Backup)

### Monitoramento do Servidor VMware
- Verificar se todas as VMs estão executando adequadamente
- **Se as VMs estiverem inativas:**
  - Verificar logs da VM para mensagens de erro
  - Identificar causa raiz (restrições de recursos, travamentos, atualizações)
  - Reiniciar VMs afetadas seguindo a sequência de inicialização adequada
  - Monitorar estabilidade após reinicialização
  - Documentar quaisquer padrões ou falhas recorrentes

### Armazenamento Remoto de Backup no OneDrive
- Verificar uso total de armazenamento
- **Se aproximando do limite de 1TB:**
  - Navegar até a pasta "Copias"
  - Revisar arquivos por data e tamanho
  - Excluir cópias de backup mais antigas ou redundantes
  - Verificar se os backups críticos permanecem intactos
  - Considerar configurar agendamento de limpeza automatizada se isso for recorrente

### Gerenciamento da Frota de Impressoras
- Verificar todas as impressoras para alertas de status
- Monitorar níveis de papel em todos os dispositivos
- Verificar níveis de cartuchos de tinta/toner
- **Se os suprimentos estiverem baixos:**
  - Reabastecer bandejas de papel conforme necessário
  - Substituir cartuchos baixos ou vazios
  - Atualizar log de inventário de impressoras
- **Se os suprimentos reserva estiverem baixos:**
  - Verificar níveis de estoque atuais contra inventário padrão
  - Gerar pedido de suprimentos (incluir cartuchos de toner, frascos de tinta, papel)
  - Submeter solicitação de compra ao departamento apropriado

## Período de Suporte Flexível (Restante do Dia)

### Protocolo de Resposta a Problemas
Problemas podem variar de **1 minuto a 2+ horas** dependendo da complexidade.

**Vitórias Rápidas (1-15 minutos):**
- Redefinições de senha
- Solução de problemas de impressora
- Problemas básicos de software
- Ajustes de conta de usuário

**Complexidade Média (15-60 minutos):**
- Instalações de software
- Problemas de conectividade de rede
- Problemas de configuração de e-mail
- Solicitações de recuperação de arquivos

**Problemas Complexos (1-2+ horas):**
- Falhas de hardware que requerem diagnóstico
- Problemas de servidor
- Problemas de infraestrutura de rede
- Tarefas de migração de dados
- Problemas de integração de múltiplos sistemas

### Melhores Práticas Durante o Suporte
- Priorizar por urgência e impacto nos negócios
- Comunicar prazos estimados aos usuários afetados
- Documentar soluções para problemas recorrentes
- Escalar quando necessário
- Fazer anotações breves para revisão ao final do dia
